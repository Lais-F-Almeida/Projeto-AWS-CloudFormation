# Laboratório AWS CloudFormation

## Sobre o projeto
Este repositório contém a prática realizada durante o laboratório de **AWS CloudFormation**, com o objetivo de criar e gerenciar recursos na nuvem por meio de **Infraestrutura como Código (IaC)**.

Durante o exercício, foram criadas stacks utilizando templates no formato **YAML**, que automatizam a criação de serviços como:
- Instância EC2 simples  
- Servidor Apache  
- Regras de firewall  
- Bucket S3  

---

## O que é o AWS CloudFormation
O **AWS CloudFormation** é um serviço que permite **provisionar e gerenciar recursos da AWS automaticamente** a partir de arquivos de template (YAML ou JSON).  
Com ele, é possível descrever toda a infraestrutura de forma declarativa, versionar configurações e reproduzir ambientes com facilidade.

**Principais benefícios:**
- Automação da criação e exclusão de recursos  
- Reutilização de templates  
- Controle de versão e rastreabilidade  
- Redução de erros manuais  
- Padronização de ambientes
